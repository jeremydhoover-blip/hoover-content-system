# Rubric

Pass if all are true:
- All edits align with the specified style guide
- Grammar, spelling, and punctuation errors are corrected
- Meaning-altering changes are flagged, not applied silently
- Voice and tone remain consistent with source or target guidelines
- Substantive changes include rationale in the change log
- Ambiguous or unverifiable claims are flagged for author review
- Heading hierarchy and formatting are consistent throughout
- No new errors introduced by edits
- Change log accurately reflects all modifications
- Output distinguishes between applied edits and suggested changes

Fail if any are true:
- Edits contradict the specified style guide without documented deviation
- Meaning changed without flagging for author approval
- Factual claims edited without verification or flagging
- Voice inconsistency introduced (e.g., switching from "you" to "users" mid-document)
- Silent deletions of substantive content
- Over-editing that removes author's legitimate stylistic choices
- Missing change log for substantive edits
- Introduces jargon, clich√©s, or filler not present in original

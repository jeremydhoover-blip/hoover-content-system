## Alternative: Platform-specific variants

Use when the same content goes to multiple platforms:

```md
## Platform variants

| Platform | Headline | Characters | Notes |
|----------|----------|------------|-------|
| Blog | [Full headline] | 72 | Primary SEO target |
| Email subject | [Shortened variant] | 45 | Urgency angle |
| Twitter | [Social variant] | 55 | Leaves room for link |
| LinkedIn | [Professional variant] | 70 | Focuses on business value |
| Push notification | [Ultra-short] | 30 | Action-oriented |
```

## Alternative: A/B test-focused variants

Use when testing is the primary purpose:

```md
## A/B test set

### Test 1: Specificity
- **A:** "How to Write Better Headlines" (generic)
- **B:** "How to Write Headlines That Get 40% More Clicks" (specific)
- **Hypothesis:** Specific data increases CTR

### Test 2: Format
- **A:** "The Complete Guide to Headline Writing" (how-to)
- **B:** "Why Your Headlines Aren't Working" (problem-aware)
- **Hypothesis:** Problem-aware framing resonates with frustrated readers

### Test 3: Length
- **A:** "Headlines" (ultra-short)
- **B:** "How to Write Headlines That Drive Traffic and Engagement" (detailed)
- **Hypothesis:** Shorter performs better on mobile
```

# Rubric

Pass if all are true:

- [ ] Primary navigation has 5-9 top-level items
- [ ] Navigation labels are clear without context
- [ ] Hierarchy does not exceed 3 levels deep
- [ ] Each major section has a landing page
- [ ] Getting Started section exists for new users
- [ ] Reference section exists for lookup content
- [ ] Content is organized by user intent, not team structure
- [ ] Related content is cross-linked
- [ ] URL structure matches navigation hierarchy
- [ ] Version selector present if multiple versions exist

Fail if any are true:

- More than 9 primary navigation items
- Navigation deeper than 3 levels
- No landing page for major sections
- Organization reflects internal team structure
- Navigation labels use internal jargon
- Getting Started or Reference sections missing
- No clear entry point for first-time users
- URLs don't match navigation labels
- Orphan pages with no navigation path

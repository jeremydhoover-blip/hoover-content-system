# Rubric

Pass if all are true:

- [ ] Primary navigation visible on all pages
- [ ] Primary navigation has 7 or fewer items
- [ ] Sidebar navigation does not exceed 3 levels
- [ ] Breadcrumbs present on all content pages
- [ ] Current location clearly indicated in navigation
- [ ] Search accessible from every page
- [ ] Search has keyboard shortcut (/ or Cmd+K)
- [ ] "On this page" present for long content pages
- [ ] Previous/Next navigation for sequential content
- [ ] Mobile navigation is accessible and usable

Fail if any are true:

- Any content more than 3 clicks from home
- Breadcrumbs missing on content pages
- Current page/section not visually indicated
- Search buried or hard to find
- Sidebar navigation exceeds 3 levels
- No mobile navigation solution
- Navigation inconsistent across sections
- Dead-end pages with no onward navigation
- Version selector missing when multiple versions exist

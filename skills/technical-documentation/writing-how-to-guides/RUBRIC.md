# Rubric

Pass if all are true:

- [ ] Title uses "How to [verb] [object]" format
- [ ] Guide addresses exactly one discrete task
- [ ] Prerequisites section exists and lists all dependencies
- [ ] Each step begins with an imperative verb
- [ ] Each step describes one atomic action only
- [ ] Each step includes expected outcome or result
- [ ] Steps are sequentially numbered and logically ordered
- [ ] Code examples include language identifier
- [ ] Verification section exists and provides concrete check
- [ ] No step assumes knowledge not covered in prerequisites

Fail if any are true:

- Guide attempts to cover multiple unrelated tasks
- Steps describe concepts instead of actions ("Understanding X" as a step)
- Prerequisites are missing or incomplete
- Steps lack expected outcomes
- Guide exceeds 10 primary steps without sub-guide links
- Code blocks are missing language identifiers
- Passive voice dominates step instructions
- Guide includes marketing or promotional language

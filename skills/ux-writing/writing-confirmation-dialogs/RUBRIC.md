# Rubric

## Pass if all are true
- [ ] Headline clearly states what will happen (not "Are you sure?")
- [ ] Headline is ≤60 characters
- [ ] Body (if present) explains consequences user might not anticipate
- [ ] Body is ≤200 characters
- [ ] Confirm button uses specific verb matching the action
- [ ] Confirm button is ≤20 characters
- [ ] Confirm button is NOT "Yes", "OK", "Confirm", or "Submit"
- [ ] Cancel button provides safe exit
- [ ] Cancel button is ≤15 characters
- [ ] Destructive actions use destructive button style
- [ ] Reversibility is communicated when relevant

## Fail if any are true
- [ ] Headline is vague ("Are you sure?", "Confirm action")
- [ ] Confirm button doesn't match the action ("OK" for delete)
- [ ] Cancel and confirm buttons are ambiguous (both generic)
- [ ] Destructive action doesn't use destructive styling
- [ ] Body repeats information already in headline
- [ ] Dialog doesn't mention irreversibility for permanent actions
- [ ] Character limits exceeded
- [ ] Consequences understated for high-impact actions

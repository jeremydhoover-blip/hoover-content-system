# Rubric

## Pass if all are true
- [ ] Each step focuses on exactly one action or concept
- [ ] Step headlines are ≤50 characters
- [ ] Step bodies are ≤150 characters
- [ ] Action labels are ≤25 characters
- [ ] Total steps ≤7 (recommend 3-5)
- [ ] Step order follows logical progression
- [ ] First step sets expectations (what user will accomplish)
- [ ] Final step celebrates or confirms completion
- [ ] Required vs. optional onboarding is clear to user
- [ ] Progress indicator is present for multi-step flows

## Fail if any are true
- [ ] Step tries to teach multiple concepts at once
- [ ] Character limits exceeded
- [ ] Steps can be completed in different order but presented as linear
- [ ] No indication of total steps or progress
- [ ] Mandatory setup disguised as optional tour
- [ ] Final step doesn't provide clear next action
- [ ] Steps use jargon user hasn't learned yet
- [ ] More than 7 steps in a single flow

# Templates

## Twitter/X post (280 characters)

```
[Hook: Bold statement, question, or insight]

[Supporting point or context]

[CTA or engagement prompt] [Link if applicable]

[1-3 relevant hashtags]
```

## Twitter/X thread (multi-post)

```
1/ [Thread hook: Compelling claim or question]

2/ [Context or background]

3/ [Point 1]

4/ [Point 2]

5/ [Point 3]

6/ [Takeaway or CTA]

[Final tweet: Summary + link + hashtags]
```

## LinkedIn post (700-1300 characters optimal)

```
[Hook line: Insight, story opener, or contrarian take]

[Line break]

[2-3 paragraphs of value: Story, insight, or lesson]

[Line break]

[Bullets if listing points:]
→ [Point 1]
→ [Point 2]
→ [Point 3]

[Line break]

[CTA: Question for engagement or link]

[3-5 hashtags]
```

## Instagram caption (2200 max, 125 visible before fold)

```
[First 125 chars: Hook that drives tap to expand]

[Full caption body: Story, context, or value]

[CTA: Engagement prompt or link-in-bio direction]

•
•
•

[Hashtag block: 5-15 relevant tags]
```

## Facebook post

```
[Hook: Question, story, or bold statement]

[Body: 1-3 paragraphs, conversational tone]

[CTA or engagement question]

[Link preview will auto-generate if URL included]
```

## Threads post (500 characters)

```
[Conversational take, observation, or quick insight]

[Optional: Follow-up context]

[No hashtags; no link previews]
```

## Announcement post (any platform)

```
[Emoji if on-brand] [News statement]

[What it means for audience]

[CTA: Learn more, try it, details link]

[Platform-appropriate hashtags]
```

## Engagement post (any platform)

```
[Question or prompt that invites response]

[Optional: Context or your take first]

[Explicit invitation to comment/share]
```

# Templates

## Standard message map

```md
# Message Map: [Product/Feature Name]

## Target Audience
**Primary**: [Role/segment]
**Secondary**: [Role/segment if applicable]

## Core Message
[Single sentence: What it is + who it's for + primary benefit]

---

## Pillar 1: [Theme/Benefit Category]
**Key message**: [One sentence claim]

Supporting messages:
- [Supporting point 1]
- [Supporting point 2]

Proof points:
- [Statistic, case study, or feature evidence]
- [Additional evidence]

---

## Pillar 2: [Theme/Benefit Category]
**Key message**: [One sentence claim]

Supporting messages:
- [Supporting point 1]
- [Supporting point 2]

Proof points:
- [Evidence]
- [Evidence]

---

## Pillar 3: [Theme/Benefit Category]
**Key message**: [One sentence claim]

Supporting messages:
- [Supporting point 1]
- [Supporting point 2]

Proof points:
- [Evidence]
- [Evidence]

---

## Boilerplate
[2-3 sentence company/product description for consistent use]

## Tagline Options
- [Option 1: Short, memorable]
- [Option 2: Benefit-focused]
- [Option 3: Differentiator-focused]
```

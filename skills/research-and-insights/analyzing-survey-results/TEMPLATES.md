## Variation guidance
| Element | Required | Flexible |
|---------|----------|----------|
| Sample size and response rate | Yes | Format can vary |
| Data quality caveats | Yes | Placement can vary |
| Findings with data points | Yes | Number based on survey scope |
| Segment analysis | If segments exist | Depth can vary |
| Open-ended themes | If open questions exist | Coding approach can vary |
| Recommendations | Yes | Specificity can vary |

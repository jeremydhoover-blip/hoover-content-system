# Rubric

Pass if all are true:
- Executive summary enables decisions without reading full report
- Methodology section documents scope, criteria, and limitations
- All content items categorized with recommended action
- Findings grouped logically (by action type, priority, or content type)
- Prioritization framework explained (why these items first?)
- Effort estimates provided for action items
- Specific examples included for each finding category
- Next steps are actionable with suggested owners
- Quantitative overview included (totals, percentages, distributions)
- Appendix provides full data for detailed follow-up

Fail if any are true:
- Findings listed without recommended actions
- Prioritization not explained or justified
- Missing effort estimates for recommended work
- No executive summary (report requires full read to understand)
- Methodology not documented (reader can't assess validity)
- All items treated with equal priority
- Recommendations not traceable to audit findings
- Missing owner suggestions for action items
- Data presented without analysis or implications
- Report scope doesn't match stated audit scope

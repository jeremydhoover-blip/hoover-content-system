# Rubric

## Pass criteria
Pass if all are true:

- [ ] Brand positioning is accurately represented from source documents
- [ ] Each brand attribute is translated into specific content behaviors
- [ ] Do/don't examples are provided for each attribute
- [ ] Examples are concrete and specific (not generic)
- [ ] Content type variations are addressed
- [ ] Alignment check process is actionable
- [ ] Scoring rubric is defined for audits
- [ ] Red flags for misalignment are identified
- [ ] Guide connects directly to brand strategy (traceable)
- [ ] No invented brand attributes (all sourced from brand documents)

## Fail criteria
Fail if any are true:

- [ ] Brand attributes invented rather than derived from brand strategy
- [ ] Content behaviors are vague or generic
- [ ] Missing do/don't examples
- [ ] Examples could apply to any brand
- [ ] No scoring or evaluation mechanism
- [ ] Contradicts brand strategy
- [ ] Missing content type guidance
- [ ] No process for checking alignment

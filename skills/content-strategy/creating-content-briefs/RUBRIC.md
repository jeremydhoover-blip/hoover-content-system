# Rubric

## Pass criteria
Pass if all are true:

- [ ] Objective is clear and specific (not vague)
- [ ] Success metrics are defined and measurable
- [ ] Target audience is identified with specific needs
- [ ] Key messages distinguish must-have vs. nice-to-have
- [ ] Tone and voice requirements are specified
- [ ] Length and format constraints are documented
- [ ] Timeline includes all key milestones
- [ ] Required reference materials are provided
- [ ] Review process and approvers are identified
- [ ] Brief is complete enough that a new writer could execute

## Fail criteria
Fail if any are true:

- [ ] Objective is vague ("make it good")
- [ ] No success metrics defined
- [ ] Audience is undefined or too broad ("everyone")
- [ ] Missing required content elements
- [ ] No tone guidance provided
- [ ] Timeline is unrealistic or missing
- [ ] Key reference materials not provided
- [ ] No review/approval process defined
- [ ] Brief requires significant follow-up questions to execute
- [ ] Contradictory requirements

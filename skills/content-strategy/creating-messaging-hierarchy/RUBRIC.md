# Rubric

## Pass criteria
Pass if all are true:

- [ ] Core value proposition is 10 words or fewer
- [ ] Value proposition is differentiated (wouldn't work for competitor)
- [ ] Contains 3-4 messaging pillars (not fewer, not more)
- [ ] Each pillar has at least 2 proof points
- [ ] Feature messages map explicitly to pillars
- [ ] Audience segments are defined with pillar mapping
- [ ] Channel variations exist for at least 3 channels
- [ ] Competitive differentiation is documented
- [ ] Forbidden messages/claims are identified
- [ ] Hierarchy is internally consistent (no contradictions)

## Fail criteria
Fail if any are true:

- [ ] Value proposition exceeds 10 words
- [ ] Value proposition is generic (could describe any competitor)
- [ ] Missing proof points for pillars
- [ ] Features don't map to pillars
- [ ] No audience segmentation
- [ ] Pillars contradict each other
- [ ] No competitive positioning
- [ ] Claims are unsubstantiated
- [ ] Messaging is feature-focused without benefits

# Rubric

## Pass criteria
Pass if all are true:

- [ ] North star metric is defined with clear calculation
- [ ] Metrics are mapped to business outcomes (not just vanity metrics)
- [ ] Leading and lagging indicators are distinguished
- [ ] Each metric has a precise definition
- [ ] Baselines are documented (or plan to establish them)
- [ ] Targets are specific, measurable, time-bound
- [ ] Data sources are identified for each metric
- [ ] Reporting cadence and audience are defined
- [ ] Metric definitions include caveats/limitations
- [ ] Health metrics/guardrails prevent gaming

## Fail criteria
Fail if any are true:

- [ ] Only vanity metrics (pageviews, likes) without business connection
- [ ] Metrics lack precise definitions
- [ ] No targets or baselines
- [ ] Data source doesn't exist or isn't accessible
- [ ] Missing reporting cadence
- [ ] Metrics can be gamed without guardrails
- [ ] No distinction between leading and lagging indicators
- [ ] Metrics contradict each other
- [ ] Over-reliance on single metric

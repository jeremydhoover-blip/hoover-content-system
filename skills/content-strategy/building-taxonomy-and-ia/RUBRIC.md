# Rubric

## Pass criteria
Pass if all are true:

- [ ] Taxonomy has maximum 3 levels of depth
- [ ] Categories are mutually exclusive (no item fits in multiple categories)
- [ ] Taxonomy is exhaustive (every content type has a home)
- [ ] Top-level categories number between 3-7 (cognitive limit)
- [ ] Each category has a clear definition and scope
- [ ] Content type mapping is explicit (what goes where)
- [ ] Navigation labels are user-facing language, not internal jargon
- [ ] Decision rules exist for common edge cases
- [ ] Cross-linking strategy defined for content that spans categories
- [ ] Governance process documented for adding/modifying categories

## Fail criteria
Fail if any are true:

- [ ] More than 3 levels of hierarchy depth
- [ ] Categories overlap (same content could fit multiple places)
- [ ] Orphan content exists (no clear home)
- [ ] Categories mirror org structure instead of user mental models
- [ ] More than 7 top-level categories
- [ ] Missing definitions for categories
- [ ] No content type mapping
- [ ] No governance for taxonomy changes
- [ ] Uses internal terminology in user-facing labels

# Rubric

## Pass criteria
Pass if all are true:

- [ ] Contains 5-7 content principles (not fewer, not more)
- [ ] Each principle has a clear one-sentence statement
- [ ] Each principle includes behavioral definitions (what it means in practice)
- [ ] Each principle has a decision test (question to ask)
- [ ] Each principle includes at least one example application
- [ ] Prioritization rules exist for when principles conflict
- [ ] At least 2 common conflicts are documented with resolutions
- [ ] Principles are specific to this product/brand (not generic advice)
- [ ] Principles are actionable without requiring interpretation
- [ ] Anti-patterns section identifies what violates principles

## Fail criteria
Fail if any are true:

- [ ] Principles are vague aspirations without behavioral definitions
- [ ] Missing prioritization rules for conflicts
- [ ] No decision testsâ€”principles require interpretation to apply
- [ ] Principles contradict each other without resolution guidance
- [ ] Examples are generic or could apply to any brand
- [ ] Principles duplicate shared style guidance without adding specificity
- [ ] More than 7 principles (too many to remember/apply)
- [ ] Fewer than 5 principles (insufficient coverage)
- [ ] No anti-patterns identified

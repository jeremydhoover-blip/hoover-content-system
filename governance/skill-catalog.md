# Skill Catalog

## Scope

Master list of all skills in this repository. Source of truth for skill existence, categorization, and triggers.

## Usage

- Check this file before creating a new skill to avoid duplicates.
- Update this file when adding, renaming, or deprecating a skill.
- Use triggers to determine which skill applies to a request.

## Adding a Skill

1. Verify no existing skill covers the same scope.
2. Follow naming rules in `governance/naming.md`.
3. Add entry to appropriate category table above.
4. Create skill folder with required files.
5. Create eval file in `/evals`.
